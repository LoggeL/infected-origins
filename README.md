## Figuresn

- **Richard Voss** — Arzt, Familiervater. Sein Fall ist moralische Erosion, nicht plötzliches Bösewerden.
- **Eva Voss** — Seine Frau. Sterbenskrank. Moralisch die stärkste Figur.
- **Nora Voss → Lilith** — Tochter. Die Hauptfigur. Beginnt als Helferin, wird zur Kontrollfigur.
- **Kessler** — GenetiX-Verhandler. Der subtilste Antagonist.
